# Sistema de Controle de Despesas

## Estratégia de Construção do Sistema

- Criar o menu principal com as opções do sistema (Entrar Despesa, Anotar Pagamento, Listar Despesas, Gerenciar Tipos de Despesa, Gerenciar Usuários e Sair).
- Cada funcionalidade será representada futuramente por *classes, atributos e métodos*, abstraindo as regras de negócio no contexto da Programação Orientada a Objetos (POO).
- A documentação de cada classe será feita dentro do README ou em arquivos separados e linkados aqui.
- Armazenamento de dados será feito em arquivos de texto separados para despesas, usuários e tipos de despesas.
